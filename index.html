🔧 备选方案：使用这个 HTML 片段嵌入任意 GIF 搜索

复制以下代码，修改 search-term 为你想要的关键词：

<iframe src="https://giphy.com/embed/search/可爱猫" width="100%" height="400" frameborder="0" allowfullscreen></iframe>
  1. 把上面代码中的 可爱猫 换成你想搜索的关键词（支持中文）
  2. 把代码粘贴到你的 HTML 文件中
  3. 保存后打开，就能看到对应关键词的 GIF 搜索结果
  4. 点击喜欢的 GIF，右键或长按保存到本地

⚡ 示例关键词：cute-cat, happy, dance, thank-you, meme, celebration

💬 在 GIPHY 中点击 GIF → 右键/长按 → "保存图片" 即可下载到设备
